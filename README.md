# 🧭 Rift Codex

Um site épico e imersivo inspirado no universo de League of Legends, criado com HTML, CSS e JavaScript puro.

O projeto simula um códice interativo de Runeterra, onde o usuário pode explorar regiões, campeões e suas histórias de forma visual e dinâmica.

---

## ✨ Preview

> Interface inspirada no universo de Runeterra, com visual cinematográfico, cores escuras e detalhes dourados.

---

## 🚀 Funcionalidades

- 🌍 Navegação por regiões de Runeterra
- 🧩 Filtro de campeões por região
- 🃏 Cards interativos de campeões
- 📖 Painel de detalhes com:
  - Lore
  - Classe
  - Lanes
  - Habilidades (Passiva, Q, W, E, R)
- 🎨 Interface épica e responsiva
- 📱 Layout adaptado para mobile
- ⬆️ Botão "voltar ao topo"
- 🎭 Animações suaves e interações

---

## 🧠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎮 Sobre o projeto

O Rift Codex foi desenvolvido como um projeto de portfólio com foco em front-end, explorando:

- manipulação de DOM
- renderização dinâmica de dados
- organização de layout
- design visual avançado
- experiência do usuário

O objetivo foi criar uma aplicação com aparência de produto real, indo além de projetos básicos.
